# Usage


### Execute
```
docker pull chj8081/skt:latest
docker run -it --rm chj8081/skt:latest
```


### Docker build
```
cd skt
docker image build -t skt .
```


# Environment
```
Python 3.7.3
Apache Spark 2.4.4
Apache Kafka 2.2.1
PostgreSQL 11.5
```
